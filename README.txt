Welcome the Ball Dodge by Jibran Khalil :)


### ABOUT ###
This game uses the Unity3D engine to create the game. C# was used in the scripts to enable things such as collision, level change and other game implementations.
The game was made following a YouTube tutorial (https://www.youtube.com/watch?v=IlKaB1etrik) but design and some functionalities were changed.

### HOW TO START ###
Unzip File 'MyFirstGame'
Open the folder 'MyFirstGame' on Unity 2019 Version.
Click the play button on top
Click start game.

### HOW TO PLAY ### 
Simply use the 'A' key to go left and 'D' key to go right! Dodge all the obstacles!
